## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/tatsmaki/todoreact.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Run development server with
   ```JS
   npm start
   ```
